# Seldon

**Systematic knowledge preservation across discontinuities.**

Seldon is a research writing and report assembly system designed for AI-assisted scholarly workflows. It orchestrates [ANTS](https://github.com/brockwebb/ai-native-traceability-system) (artifact traceability) and [Wintermute](https://github.com/brockwebb/wintermute) (knowledge vault) to produce coherent research output with proper citations across sessions.

## Problem

Research writing with AI tools fragments across conversation threads, context windows, and tool boundaries. Findings drift, citations get lost, structural coherence degrades. The more sessions involved, the worse the entropy.

## Approach

Seldon provides templated workflows, citation management, and session continuity for long-form research projects. It doesn't replace ANTS or Wintermute — it sits on top, using both.

## Status

Early development. Establishing project structure and core templates.
